This is the first post on my new weblog. I'm going to use this space to track four kinds of things:

- **Projects** I'm building or shipping
- **Blog** posts — essays, notes, observations
- **Tools** I've come across and found useful
- **Research** notes on things I'm learning

Everything shows up in a single chronological stream on the homepage, and each type has its own index page. Tags work across all types.
